Locally disabling gradient computation ​

The context manager torch.no_grad() is helpful for locally disabling and enabling gradient computation.

python
>>> x = torch.zeros(1, requires_grad=True)
>>> with torch.no_grad():
...     y = x * 2
>>> y.requires_grad
False
FunctionDescription
no_gradContext manager that disables gradient computation.

torch.abs ​

python
torch.abs(input: Tensor) -> Tensor

Computes the absolute value of each element in input.

Example

python
>>> torch.abs(torch.tensor([-1, -2, 3]))
tensor([1, 2, 3])

torch.add ​

python
torch.add(input: Tensor, other: Tensor) -> Tensor

Adds other to input.

Supports broadcasting to a common shape.

Example

python
>>> a = torch.randn(4)
>>> a
tensor([0.8899437241853971, 0.8899437241853971, 0.8899437241853971, 0.8899437241853971])
>>> torch.add(a, 20)
tensor([20.889943724185397, 20.889943724185397, 20.889943724185397, 20.889943724185397])
>>> b = torch.randn(4)
>>> b
tensor([0.11432944056866967, 0.11432944056866967, 0.11432944056866967, 0.11432944056866967])
>>> c = torch.randn(4, 1)
>>> c
tensor([[0.2104800694898007], [0.2104800694898007], [0.2104800694898007], [0.2104800694898007]])
>>> torch.add(b, c)
tensor([[0.32480951005847036, 0.32480951005847036, 0.32480951005847036, 0.32480951005847036], [0.32480951005847036, 0.32480951005847036, 0.32480951005847036, 0.32480951005847036], [0.32480951005847036, 0.32480951005847036, 0.32480951005847036, 0.32480951005847036], [0.32480951005847036, 0.32480951005847036, 0.32480951005847036, 0.32480951005847036]])
